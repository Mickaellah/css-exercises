Learning about grid layout.

Answers of the questions in index.md about grid.

  • By default, grid have one column to work on.
  • The difference between flexbox and grid are:
    1. Flexbox was invented for layout in one dimention, either a row or a column.
    2. Grid was invented for two-dimentional layout, rows and columns at the same time.
    And also flexbox can align elements along a baseline while grid layout cannot do that.
  • We would like to use flexbox rather than grid because flexbox is the best way to arrange elements in either an single row or single column.
  • We choose to use grid rather than flexbox because grit is the best way to arrange elements in multiple rows and coumns.
  • The short way to write out: grid-template-columns: 10rem 10rem 10rem 10rem; is grid-template-columns: repeat(4 10rem);.
  • To specify a minimum height for a grid row we can use the minmax() function with grid-template-row.
