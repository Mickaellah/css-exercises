Dig into transitions.

  1. Transition property is a shorthand property of transition-property, transition-duration, transition-timing-function and transition-delay.
  2. The difference between a timing function, delay and duration is
    • Timing function is the speed.
    • Delay is the duration of the waiting time.
    • Duration is the length of time it will take.
  3. In CSS, all the properties that are using any numeric values can be animated by using transition.
  4. We can select more than one property to transition at a time by using the all keyword or putting comma to separate them.
  5. To define the desire state, we use the pseudo classeslike :hover, :active, :focus, :target, :checked, or just use any selectors different from the other one.
  6. We do need transitions, and they are useful because they give a way to control animation speed when changing CSS properties.
  7. Transition became a problem when you are in a rush.
  8. Transitions don't work reliably with the auto keyword being one of the end states because animation only work with any numeric value.
  9. For us to stage transition we set the transition-delay for each single element.
