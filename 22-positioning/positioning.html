Position property.

Static is the diffault value of position property.

  • When we are using any non-default position value, we need to supply width and height first.
  • The inferred display value of an element that has been positioned is block.
  • I think absolute position is the one which still maintains a box in the flow of the documen because it is said that this value creates a new stacking context when the value of z-index is not auto. The margins of absolutely positioned boxes do not collapse with other margins.
  • The something which makes fixed and sticky positioning is that they always create a new stacking context.
  • We can use top and right properties to lock an element absolutely position's coordinates to its parent element.
  • Absolute positioned layout is useful for elements that should not interact.
  • In css, stacking context is the key to understand the css z-index.
