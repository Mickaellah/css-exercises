Doing some research on how to use animation and what do we need to include in it.

  1. Keyframe must start with the @ rule.
  2. Keyframes are different to the rest of CSS because it can specify the starting and the ending of the animation and also in media queri we have the round brackets, but instead we use curly braces in keyframe.
  3. To use the keyframe, we have to create a @keyframes rules with a name that is then used by the animation-name property, and to apply it with an element, we call the animation-name and animation-duration.
  4. Animation have other keywords that aren't available in transition like
    • reverse
    • infinite
    • alternate
    • alternate-reverse
    • play state
    • running
    • paused
    • fill mode
  5. We can animate as many properties as we want if only the properties are animatable.
  6. All elements can be animated including pseudo-elements.
