1. Keyframe starts with the @ symbol.
  2. Keyframes are different to the rest of CSS because it can specify the starting and the ending of the animation.
  3. To use the keyframe, we have the 0% which means start and 100% which means end.
  4. Animation have other keywords that aren't available in transition like
    • Name.
    • Direction.
    • Iteration.
    • Play state.
    • Fill mode.
  6. All elements can be animated including pseudo-elements.
