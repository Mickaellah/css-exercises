First part (part1) of this exercise.

  1. Cursor property is used to set what kind of cursor to show when the mouse pointer is over an element.
  2. The pointer-event property sets under what circumstances a particular graphic element can become the target of pointer events.
  3. Outline is a line that is drawn around an element, it sets outide the borders, and it is used to make the element easy to differ from the others.
  4. Filter property applies some graphical effects like blur, shadow or color shift to an element.
  5. The appearance property is used to display an element using a platform-native styling based on the user's operating system's them.
  6. The object-fit property sets how the content of a replaced element, such as img or video, should be resized to fit its container.
  7. The scroll-snap-align property is used to specify the box's snap position as an alignement of its snap area within its snap container's support.

Second part (part2) of this exercise, and this is about pseudo classes.

  1. :focus pseudo class represents an element that has received focus. It is happen when the user clicks or taps on an element or selects it with the keyboard's tab key.
  2. :focus-within pseudo class represents an element that is itself matched by the :focus pseudo-class or has a descendant that is matched by :focus.
  3. :visited pseudo class shows links that user has been visited.
  4. :active represents an element thas is being activated by the user. When using a mouse, "activation" typically starts when the user presses down the primary mouse button.
  5. :target pseudo class represents a unique element with an id (identification) matching the URL's fragement.
  6. :disabled pseudo class represents any disabled element. Disabled elements are those elements that can't be activated (selected, clicked on, typed into, etc) or accept focus.
  7. :empty pseudo class represents any element that has no children.
  8. :not() pseudo class represents elements that do not match a list of selectors. It is known as the negation pseudo-class.
  9. :matches() pseudo class checks whether the element at its position in the outer selector matches any of the selectors in its selector list. This works in the same way as :is() and :any() work.

Pseudo element.

The ::first-letter pseudo element puts in some styles to the first letter of the first line of a block-level-element, but only when not preceded by other content.

A mysterious @-block.

The @support block allows you to specify declarations that depend on a browser's support for one or more specific CSS features. This is called a feature query. This rule can be place at the top of your code or nested inside any other conditional group at-rule.

Pseudo element value function that has a promising future.

attr() is used to retrieve the value of an attribute of the selected element and use it in the stylesheet. It can also be used on pseudo-elements, in which case the value of the attribute on th pseudo-element's originating element is returned.
