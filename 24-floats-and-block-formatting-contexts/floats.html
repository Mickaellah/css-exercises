Understanding floats and block formatting context.

Floats

  1. The difault value of float and clear is none.
  2. Clear property have quite lots of possible values:
    • none
    • both
    • left
    • right
    • inline-start
    • inline-end
    • inherit
  3. The rule associated with a floated item in terms of source order is that the floated element must comes first, before the others.
  4. Yes, we can float other elements besides images.
  5. I think we can us float to create a two-column layout.

BFC (block formatting context)

  1. BFC is all about layout of block boxes in which floats interact with other elements. It is set of rules on how container should contain its contents.
  2. We can clear a float by creating a block formatting context on the parent, this is achieved by putting a certain set of properties on the container. Those are the properties we need to use with that:
    • position: absolute;
    • display: flow_root;
  3. I do not know yet.
