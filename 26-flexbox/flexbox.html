Learning about flexbox and all the properties that have relation with it.

Answers of the questions in index.md about flexbox.

  1. The order of property values in the shorthand flex property is: flex-grow, flex-shrink, flex-basis, here in this example 2 means flex-grow, 1 means flex-shrink and auto means flex-basis.
  2. Before we use align-content, we have to use flex-wrap or flex-flow.
  3. The difference between align-content and align-items is:
    • Align-content is similar to how justify-content aligns individual items within the main-axis.
    • Align-items defines the default behaviour for how items laid out along the cross-axis.
  4. The flex item's align-self overrides the other.
  5. The box-sizing property in box model is similar to flex-basis property in flex.
  6. The other property that can change the whole order of the flex items at the container level is flex-direction with its value row-reverse or column-reverse.
  7. To create space between each item in a flex container, we can use the space-between value to create space between the items and also using the space-around value to create space around the items in a flex container.
