Learning more about the calc() function in css.

  1. For my understanding, a calc() function supports mathematical expression, strings, fractions as its value.
  2. For us to control a mathematical order of operation (PEDMAS / BODMAS) in the function like the following example calc(2 + 3 * 4) equal to 14 while it also could be equal to 20, we rewrite the calc() function like this calc((2 + 3) * 4) to give the result 20.
  3. I am bot sure that we can use cal() function to convert one type of unit to one another type of unit.
  4. If we want to convert a unitless number to a unit value, we have to multiply the unitless number by 1 with the unit which we want to use.
  5. The plus and minus operators inside the function need spaces around them because the number after the plus or minus sign will be considered as just as a negative or positive number.
  6. To convert a value into a negative value, we multiply the value by a negative number like -1.
  7. We can't use calc() function with colour values.
