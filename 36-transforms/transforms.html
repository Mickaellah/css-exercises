More about transform property.

  1. In a transform's value, we can use
    • scale()
    • rotate()
    • translate()
    • skew()
    • perspective()
    • matrix()
  2. we have to use separate functions, in order, and as values because problems may appear if your function is not in order, an also the first will be ignored, the second transform will override the first one.
  3. When we want to move the central pin or axis of the transform away from its default position.we use transform-origin to do that.
  4. We use tranform: translate(); rather than position because
    • transform: translate(); is not involved layout and have better performance.
    • position is involved layout.
  5. When an element is translated / transformed, it doesn't affect the flow of the document or other element's flow, it occurs outside of the document's flow. it is quite simiar to flex layout method.
  6. When overriding a transform in a more specific selector, problems can be seen because the transform which you set the other selector will be ignored.
  7. If we want to go 3D,we need to use transform: matrix3d() property to set the view over the scene.
  8. We can control what the transform uses in its calculations in terms of the box model. To adjust that we need to use the transform-box.
