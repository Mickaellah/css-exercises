Doing some research about Custom Property.

  1. When difining a Custom Property's name, we have to use two dashes (--) in front of the name that we are going to define, any alphanumeric ,underscore (_), emojis also can be use as part of the name if that are needed.
  2. A Custom Property's name is the same as the othe properties, we can't use spaces, instead we use dashes and also there can't be period (.) in there.
  3. There are possible values that we can assign to a new Custom Property like numbers, texts, keywords, etc.
  4. When you want to apply the Custom Property to all the elements in a document, it is good to use :root or html selector and all the elements will inherit from it, but when you want to apply the Custom Property to a specific element, it is not good to use it with the :root or html.
  5. After we've defined a new Custom Property, we use the var() function to recall or to use that Custom Property again.
  6. There are some ways that we can change a Custom Property, we can do that by editing the name, type or its value, by using media querie.
  7. Custom Properties are useful because it allows CSS variable to be used in media queries, and also to write less codes, clean codes and to simplify them.
  8. The only aspect (or part) of CSS syntax that we can use Custom Properties for is the declaration block.
