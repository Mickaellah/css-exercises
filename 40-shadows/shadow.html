Looking at a new property that is Text-shadow and Box-shadow.

  1. The difference between text-shadow and box-shadow is that with:
    • Text-shadow, we cannot apply the inset keyword and also we cannot have thickness with it.
    • Box-shadow, we can have both inset keyword and also thickness value.
  2. If we wanted to create a hard-edge shadow, we have to add the thickness with a big number as a value.
  3. Yes, it is possible to create a shadow on a circular element, but we can create in a polygon.
  4. Yes, it is possible to create multiple shadows on the same element by using comma to separate the two values and negative values with the second one.
